# CaptionAI: AI Powered Social Media Caption Generator

## Features
- AI Caption Generator
- Hashtag Generator
- CTA Generator
- Download TXT/PDF
- History Management
- Dark Mode
- Responsive UI

## Technologies Used
- Python
- Flask
- Groq API
- HTML
- CSS
- JavaScript
- SQLite
- Ngrok

## Installation
1. Clone repository
2. Create virtual environment
3. Install requirements
4. Add GROQ_API_KEY to .env
5. Run python app.py

## Future Scope
- Multi-language support
